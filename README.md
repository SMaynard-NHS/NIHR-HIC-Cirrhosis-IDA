# NIHR-HIC-Cirrhosis-IDA
Machine learning to classify iron deficiency anaemia in patients with cirrhosis within the NIHR Liver Disease Health Informatics Collaborative dataset.

This project code was run in R 4.4.3 and was part of my DPhil thesis on the use of routinely collected data for patient blood management in liver cirrhosis.

## Citation

If you use MLIDA in your work, please cite:

Maynard, S., *et al*. (2026). *Machine Learning to Classify Iron Deficiency Anaemia in Cirrhosis using Routinely Collected Data*.
